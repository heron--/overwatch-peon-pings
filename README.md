# Overwatch Sound Packs for peon-ping

A complete collection of [peon-ping](https://github.com/PeonPing/peon-ping) sound packs for every Overwatch hero.

## Install

```bash
peon packs use <character>
```

For example: `peon packs use moira`, `peon packs use tracer`, `peon packs use reinhardt`

## Heroes

### Tank (14)
Ana, Domina, Doomfist, Hazard, Junker Queen, Mauga, Orisa, Ramattra, Reinhardt, Roadhog, Sigma, Winston, Zarya

### Damage (21)
Ashe, Bastion, Cassidy, Echo, Emre, Freja, Genji, Hanzo, Junkrat, Mei, Pharah, Reaper, Sojourn, Soldier: 76, Sombra, Symmetra, Torbjorn, Tracer, Vendetta, Venture, Widowmaker

### Support (10)
Ana, Baptiste, Brigitte, Illari, Jetpack Cat, Juno, Lifeweaver, Lucio, Mercy, Mizuki, Zenyatta

Each hero has ~45 sounds across 9 CESP event categories (session.start, task.acknowledge, task.complete, task.error, input.required, resource.limit, user.spam, session.end, task.progress).
